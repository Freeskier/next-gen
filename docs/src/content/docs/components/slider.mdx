## Usage

<Tabs>
	<TabItem label="Builder">
```svelte
<script lang="ts">
	import { Slider } from "melt/builders";

	const slider = new Slider();
</script>

<div {...slider.root}>
	<div {...slider.track}>
		<div {...slider.range}></div>
	</div>
	<div {...slider.thumb}></div>
</div>
```
	</TabItem>

	<TabItem label="Component">

```svelte
<script lang="ts">
	import { Slider } from "melt/components";
</script>

<Slider>
	{#snippet children(slider)}
		<div {...slider.root}>
			<div {...slider.track}>
				<div {...slider.range}></div>
			</div>
			<div {...slider.thumb}></div>
		</div>
	{/snippet}
</Slider>
	```
	</TabItem>
</Tabs>

### Positioning

By default, Melt does not provide styles. This means that you'll be responsible for positioning the
thumb and range of your slider.

To facilitate this, Melt provides some CSS properties.
